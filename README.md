# Covid-19-Database-GeoJSON
A Covid-19 database and GeoJSON spread map, made in Python using PySimpleGUI, updated every day.

## Requirements
- Python 3.7.3 and up
- matplotlib
- PySimpleGUI


#### How to start
Execute:
`covid-19-spread.py`

Command line:
`python (path_to_file)/covid-19-spread.py`
